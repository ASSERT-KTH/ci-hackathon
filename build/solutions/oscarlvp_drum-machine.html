The Travis CI Drum Machine

This simmple web application generates sound from the events fired by Travis CI. The application implements a virtual drum machine with 47 instruments (rows) and 53 slots for each instrument (columns).

When an event occurs, the repository identifier is used to compute a cell in the machine and the value of that cell is changed.

This project uses the awesome p5js and the incredible WebAudioFont libraries.

The drum machine code is based on this example from WebAudioFont.

# Piano-Hero


Game Piano-Hero with HTML5.

[Documentation](doc/piano-hero.pdf)


## Prerequires

1. Git 2.6+
3. Browser


## How to Play

Clone

```
git clone https://github.com/humbertodias/game-piano-hero-html5.git
```

Inside the folder

```
cd game-piano-hero-html5
```

Now, open **index.html** on your favorite browser.
